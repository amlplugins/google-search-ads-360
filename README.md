# @amlplugins/google-search-ads-360

Native `@googleapis/searchads360` SDK re-exported under the `@amlplugins` namespace.

> Search Ads 360.

## Install

```bash
npm install @amlplugins/google-search-ads-360
```

## Use

```ts
import { /* named exports from @googleapis/searchads360 */ } from "@amlplugins/google-search-ads-360";
// or
import sdk from "@amlplugins/google-search-ads-360";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
